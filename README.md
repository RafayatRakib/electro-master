
Clone the Repo:

git clone https://github.com/RafayatRakib/electro-master.git

cd electro-master

composer install or composer update

cp .env.example .env

Set up .env file

php artisan key:generate

php artisan migrate:fresh --seed

php artisan serve

http://127.0.0.1:8000/

php artisan queue:work
